# Aplicao-em-Python
Proposta:
O desafio era criar um site de música em python utilizando o framework django, criando um CRUD e utilizando o admin do django para controlar as informações do sqlite. O CRUD deveria enviar informações como: nome da música, nome do artista, data da inscrição, para o sqlite.

Processo de Criação:
Como consegui todo o conhecimento de python em 1 semana?
Primeiramente pesquisei o que seria o django e como utilizá-lo instalá-lo, e logo no primeiro dia instalei e comecei a desenvolver meu front-end, de início tive a ideia de criar um front-end com navbar e no mínimo 4 páginas, a ideia era fazer um site completo com um design muito bem detalhado, mas este pensamento começou a conflitar com a ideia de fazer um site simples mas bem estruturado, e minha decisão foi a segunda opção, já que seria mais analisado o back-end.

Durante a criação tive várias dúvidas sobre como lançar os models no sqlite, mas todas foram tiradas com pesquisas em sites como stackOverFlow, e até tutoriais no youtube, desenvolvi e lancei no banco de dados com sucesso mas infelizmente não consegui terminar meu CRUD, pois não consegui resolver os erros que a aplicação apresentava quando era executado o formulário.

Infelizmente o site não está completo mas creio que pude aprender muito sobre python e django, creio que este desafio me despertou um grande interesse em python e aplicações com o django.
